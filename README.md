# How to install

After cloning, run 

```bash
python3 -m venv venv
source venv/bin/activate
python3 -m pip install -r requirements.txt
python3 -m ipykernel install --user --name hackathon
```

Now you can run the modules in this repo. Open jupyter lab with

```bash
jupyter lab .
```
